HTML Tanımı

HTML (HyperText Markup Language), web sayfalarını oluşturmak için kullanılan bir işaretleme dilidir. Web tarayıcıları, HTML kodlarını okuyarak metinleri, resimleri, bağlantıları ve diğer içerikleri kullanıcıya görsel olarak sunar. HTML, temel olarak etiketler (tags) kullanılarak yapılandırılır. Her etiket, belirli bir işlevi yerine getirir ve genellikle açılış ve kapanış şeklinde kullanılır. Örneğin, etiketi bir paragrafı belirtirken, etiketi başlıkları oluşturur. HTML, tek başına bir programlama dili değildir; ancak CSS (Cascading Style Sheets) ve JavaScript ile birlikte kullanılarak interaktif ve estetik web sayfaları oluşturulabilir.

CSS Tanımı

CSS (Cascading Style Sheets), HTML ile oluşturulan web sayfalarının görünümünü ve düzenini belirlemek için kullanılan bir stil dilidir. CSS sayesinde metinlerin rengi, yazı tipi, arka plan, kenarlıklar, boşluklar ve sayfa düzeni gibi birçok görsel öğe kolayca şekillendirilebilir. CSS, HTML'den ayrı bir dosyada veya doğrudan HTML içinde tanımlanabilir. Üç temel kullanım yöntemi vardır: 1 - Satır içi CSS (Inline CSS) 2 - Dahili CSS (Internal CSS) 3 - Harici CSS (External CSS)

Bootstrap Tanımı

Bootstrap, duyarlı (responsive) ve modern web tasarımları oluşturmak için kullanılan popüler bir HTML, CSS ve JavaScript kütüphanesidir. Twitter tarafından geliştirilmiş ve açık kaynak olarak yayınlanmıştır. Bootstrap, web sitelerinin ve uygulamalarının hızlı ve kolay bir şekilde tasarlanmasını sağlar. Bootstrap’in en önemli özelliklerinden biri, grid sistemi sayesinde esnek ve mobil uyumlu (responsive) düzenler oluşturabilmesidir. Ayrıca, önceden hazırlanmış butonlar, formlar, kartlar, açılır menüler, modal pencereler ve animasyonlar gibi birçok bileşen içerir.
